HTML 이란?

* 웹표준(W3C 규칙)을 따릅니다. * HTML(Hyper Text Markup Language)은 인터넷 전용 브로우저에 그림이나 text 를 표현할 수 있도록 해 주는 언어 Hyper Text 는 Link를 사용하여 공간의 초월의미가 부여되는 글자들을 말한다. Markup이란 컴퓨터가 문서를 생성할 수 있게 하는 신호체계를 설정하는 것.

Frame 이란 ?

FORM은 정보를 입력 또는 선택하고 버튼을 클릭하며 정보를 서버에 전달하고 응답받는 양방향의 의사소통을 지원하는 입력양식이며, Form 의 내부에는 INPUT, SELECT, TEXTAREA 등의 태그들이 포함된다.

DL, DT, DD

=> DL 이란 ? 문장을 타이틀(Dt)과 내용(Dd)으로 구분하여 표시해주는 Tag 이다.
